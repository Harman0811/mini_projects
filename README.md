## Python Calculator

## Description

This is a simple terminal-based calculator built using Python. It performs basic arithmetic operations:

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)

This project was created as a beginner practice to understand:

* User input
* if-else conditions
* Loops
* Error handling

### Features

* Continuous calculations using loop
* Handles invalid input
* Prevents division by zero
* Beginner-friendly code

### Learning Purpose

This project is part of my Python learning journey.


## Number Guessing Game

### Description

This is a simple number guessing game built using Python.

* The user tries to guess the number

This project helps in understanding:

### Features

This project helped me improve logic building and working with user inputs.
