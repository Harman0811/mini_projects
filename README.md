#  Python Calculator 

##  Description

This is a simple terminal-based calculator built using Python.
It performs basic arithmetic operations:

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)

This project was created as a beginner practice to understand:

* User input
* if-else conditions
* Loops
* Error handling

##  Features

* Continuous calculations using loop
* Handles invalid input
* Prevents division by zero
* Beginner-friendly code
