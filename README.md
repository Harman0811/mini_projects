## mini_projects

A clean and modular collection of mini projects focused on performance, readability, and beginner-friendly Python concepts. Each project is simple, well-structured, and built to demonstrate core programming principles.


## Python Calculator

### Description

This is a simple terminal-based calculator built using Python. It performs basic arithmetic operations:

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)

This project was created as a beginner practice to understand:

* User input
* if-else conditions
* Loops
* Error handling

### Features

* Continuous calculations using loop
* Handles invalid input
* Prevents division by zero
* Beginner-friendly code

### Learning Purpose

This project is part of my Python learning journey.


## Number Guessing Game

### Description

This is a simple number guessing game built using Python.

* The computer generates a random number
* The user tries to guess the number
* The program gives hints (Too High / Too Low)

This project helps in understanding:

* Random module
* Loops
* Conditional statements
* User interaction

### Features

* Random number generation
* Hint system for guesses
* Tracks user attempts (if implemented)
* Simple and interactive gameplay

### Learning Purpose

This project helped me improve logic building and working with user inputs.
